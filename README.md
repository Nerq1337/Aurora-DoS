# AuroraDDoS

> DDoS Stress Tool

### Installation:
Install the module "Kira" with:

```bash
$ npm install kira
```

Install the module "Prompts" with:

```bash
$ npm install prompts
```

### How to start:

```bash
node AuroraDDoS
```

### Warning ⚠️

Do not use this module to attack servers and services you don't own! It is only for testing purposes and not for unauthorized actions.
