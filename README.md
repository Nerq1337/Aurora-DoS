# AuroraDDoS

> DDoS Stress tool

### Installation
Install the module "Kira" with:

```bash
$ npm install -g kira
```

Install the module "Prompts" with:

```bash
$ npm install prompts
```

### How to start:

```bash
node index
```

### Warning

Do not use this module ta attack servers and services you don't own! It is only for testing purposes and not for unauthorized actions.
